# Colophon
## Content
The main content on this part of the website will include random ideas, opinions, and occasionally tutorials, mostly focused on tech-related topics; with the hope that they will help you become a better developer, designer or whatever else.


## Tools
The website is built using **React** in conjunction with **React Router**; as for all the posts, they are just markdown files which are converted first into normal text, thanks to **Vite** `?raw` option, and finally to HTML thanks to the ***react-markdown*** library.
For the stylesheet I'm using CSS Modules for Scoped CSS (literally the best thing).
